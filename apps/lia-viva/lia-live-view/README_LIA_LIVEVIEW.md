
# LIA Live View - Futuristic Realtime Interface

**LIA Live View** is a cutting-edge React application that serves as the frontend for the LIA AI Agent. It leverages the **Gemini Live API** (WebRTC) for ultra-low latency audio conversations and includes a multimodal engine for generating charts, videos (Veo), and high-resolution images (Imagen) on the fly.

## ğŸ”’ Security & Backend Integration (IMPORTANT)

This project uses the **Secure Backend Strategy**. It does not hardcode API keys in production builds.

### Contract with Backend (AntiGravity / LIA Backend)
To integrate with your main LIA backend, you must expose the following endpoint:

**Request:** `GET /api/session`

**Response (JSON):**
```json
{
  "apiKey": "YOUR_GEMINI_API_KEY"
}
```

The `ConfigService` (`src/services/configService.ts`) automatically handles this negotiation:
1.  It calls `/api/session`.
2.  If successful, it uses that key for all Gemini calls.
3.  If it fails (e.g., local dev), it falls back to `process.env.API_KEY`.
