# LIA Modes

Este app conterá os 3 modos de interação com a LIA:

- **ChatMode** (Start) - Chat básico
- **Multimodal** (Plus) - Imagens, arquivos, análise
- **LiveMode** (Pro) - Voz e vídeo em tempo real
